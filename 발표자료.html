AX HACKATHON05
VALIDATION

믿을 수 있게, 이렇게 검증했습니다

  • 제공 데이터와 무관한 극단적 더미 데이터로 스트레스 테스트 → 포맷이 다른 콘텐츠가 섞여 비교되는 버그를 발견해 수정
  • 과거 69건을 하나씩 숨기고 나머지로 예측하는 leave-one-out 백테스트로 가중치 변경 효과를 검증 — 정확도 손실 없이(p=0.36) 예측 구간이 22% 좁아짐(p=0.0025)
  • 데이터 생성 규칙을 다르게 설계한 도메인 시프트 더미데이터로 같은 통계 검증을 재실행 → 유의했던/무의미했던 축이 실제로 뒤바뀌고, 참효과가 없는 축에서 우연한 통계 오류(1종 오류, p=0.0148)까지 재현
  • 전혀 다른 신규 콘텐츠 3건으로 파이프라인을 재실행해 동일 절차·품질로 재현되는지 확인
