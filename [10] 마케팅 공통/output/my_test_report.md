# 콘텐츠 성과 예측 리포트 (파이프라인 생성)

> `scripts/pipeline.py`로 자동 생성됨. ML 예측이 아닌, 유사 콘텐츠의 실제 과거 성과 기반 패턴 진단입니다.

## 1. 데이터 개요

- 과거 데이터 정제 후 69건 기준 (원본 중복·결측 포함 여부는 입력 파일에 따라 달라질 수 있음)
- 신규 진단 대상: 5건

## 2. 유사도 기준
- 앵커: `type` + `channel` 동시 일치. `type`은 원칙적으로 완화하지 않음(형식이 다르면 비교 자체가 무의미)
- 서브 점수(4점 만점): `posting_hour` 일치 +2 / `topic_category` 일치 +1 / `has_emoji` 일치 +1
- 동점 시 `content_id` 오름차순. 앵커 후보 3건 미달 시 channel만 완화(type 유지), 그래도 부족하면 있는 대로 사용
- 해당 `type`이 과거 데이터에 전혀 없는 경우에만 최후 수단으로 전체 데이터 참고(신뢰도 매우 낮음으로 표시)

## 3. 신규 콘텐츠 진단

### 신규 1: 주니어 마케터 6개월 성장기
- 조건: 숏폼 / 커리어 / 유튜브 / 20시 / 이모지 O / 제목 15자
- **예상 CTR 범위: 4.5% ~ 5.8%** (앵커 후보 13건, type+channel 앵커 유지, 최고 서브점수 4/4)

| 순위 | content_id | 제목 | CTR | 인게이지먼트율 | 유사 이유 |
|---|---|---|---|---|---|
| 1 | C002 | 주니어 마케터가 3개월 만에 한 일 | 5.2% | 8.7% | 서브점수 4/4 |
| 2 | C066 | 마케팅 직무 연봉 현실 공개 | 4.5% | 8.2% | 서브점수 4/4 |
| 3 | C001 | 퇴근 후 5분 루틴으로 인생 바뀜 | 5.8% | 9.2% | 서브점수 3/4 |

**개선 제안 2가지**
1. 제목 길이를 늘리는 실험(근거 약함, 표본 작음): 앵커 내 현재 그룹 평균 4.08%(n=5) vs 반대 그룹 4.58%(n=8), +0.50%p
2. 주제 앵글을 '생산성' 쪽으로 조정 실험(근거 약함 — topic_category는 전체 통계 검증상 CTR과 유의한 관계 없음): 앵커 내 현재(커리어) 4.40%(n=3) vs 생산성 4.50%(n=4), +0.10%p
