# 콘텐츠 성과 예측 리포트 (파이프라인 생성)

> `scripts/pipeline.py`로 자동 생성됨. ML 예측이 아닌, 유사 콘텐츠의 실제 과거 성과 기반 패턴 진단입니다.

## 1. 데이터 개요

- 과거 데이터 정제 후 50건 기준 (원본 중복·결측 포함 여부는 입력 파일에 따라 달라질 수 있음)
- 신규 진단 대상: 6건

## 2. 유사도 기준
- 앵커: `type` + `channel` 동시 일치. `type`은 원칙적으로 완화하지 않음(형식이 다르면 비교 자체가 무의미)
- 서브 점수(3점 만점): `posting_hour` 일치 +2 / `has_emoji` 일치 +1 (`topic_category`는 ANOVA 검증 결과 CTR과 유의한 관계 없어 0점 처리, decisions.md 참고)
- 동점 시 `content_id` 오름차순. 앵커 후보 3건 미달 시 channel만 완화(type 유지), 그래도 부족하면 있는 대로 사용
- 해당 `type`이 과거 데이터에 전혀 없는 경우에만 최후 수단으로 전체 데이터 참고(신뢰도 매우 낮음으로 표시)

## 3. 신규 콘텐츠 진단

### 신규 1: 퇴근길에 듣는 생산성 꿀팁
- 조건: 숏폼 / 생산성 / 유튜브 / 20시 / 이모지 O / 제목 14자
- **예상 CTR 범위: 4.9% ~ 6.0%** (앵커 후보 10건, type+channel 앵커 유지, 최고 서브점수 3/3)

| 순위 | content_id | 제목 | CTR | 인게이지먼트율 | 유사 이유 |
|---|---|---|---|---|---|
| 1 | D028 | [검증용] 숏폼-후기-유튜브-20시 샘플 28 | 6.0% | 9.4% | 서브점수 3/3 |
| 2 | D017 | [검증용] 숏폼-커리어-유튜브-8시 샘플 17 | 5.3% | 8.6% | 서브점수 1/3 |
| 3 | D018 | [검증용] 숏폼-커리어-유튜브-8시 샘플 18 | 4.9% | 6.6% | 서브점수 1/3 |

**개선 제안 2가지**
1. 주제 앵글을 '후기' 쪽으로 조정 실험(근거 약함 — topic_category는 전체 통계 검증상 CTR과 유의한 관계 없음): 앵커 내 현재(생산성) 4.70%(n=3) vs 후기 5.07%(n=3), +0.37%p
2. 핵심 변수(시간대·이모지·포맷)에서 이미 최적/준최적 구간에 있어 추가로 바꿀 요소가 제한적입니다. 그대로 발행을 권장합니다.
