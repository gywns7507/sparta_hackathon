# past_content_performance.csv 컬럼 정의서

- 파일 설명: 과거 발행 콘텐츠 70건의 성과 데이터
- 총 행 수: 70 (데이터 행, 헤더 제외)
- 총 컬럼 수: 11
- 식별자 기준 고유 건수: 69 (content_id `C003`이 2회 등장하는 중복 행 존재 → 행 3, 71)
- 결측치: `C069`의 engagement_rate, `C070`의 reach 값 없음

| # | 컬럼명 | 데이터 타입 | 설명 | 값 범위 / 예시 | 결측치 |
|---|---|---|---|---|---|
| 1 | content_id | string | 콘텐츠 고유 식별자 (형식: `C` + 3자리 숫자) | C001 ~ C070 (C068 결번, C003 중복 1건) | 없음 |
| 2 | title | string | 콘텐츠 제목 (한글) | "퇴근 후 5분 루틴으로 인생 바뀜" 등 | 없음 |
| 3 | type | string (범주형) | 콘텐츠 유형 | 숏폼 / 카드뉴스 / 블로그 / 인포그래픽 | 없음 |
| 4 | topic_category | string (범주형) | 콘텐츠 주제 카테고리 | 생산성 / 커리어 / 트렌드 / 후기 | 없음 |
| 5 | channel | string (범주형) | 발행 채널 | 유튜브 / 인스타그램 / 블로그 | 없음 |
| 6 | ctr | float | 클릭률 (%) | 1.4 ~ 5.8 | 없음 |
| 7 | engagement_rate | float | 인게이지먼트율 (%, 좋아요·댓글·공유 등 반응 비율) | 2.1 ~ 9.2 | 1건 (C069) |
| 8 | reach | integer | 도달 수 (콘텐츠에 노출된 사용자 수) | 5,900 ~ 45,100 | 1건 (C070) |
| 9 | headline_length | integer | 제목(헤드라인) 글자 수 | 12 ~ 42 | 없음 |
| 10 | has_emoji | boolean | 제목에 이모지 포함 여부 | True / False | 없음 |
| 11 | posting_hour | integer | 게시 시각 (24시간제, 시 단위) | 8 / 12 / 18 / 20 | 없음 |

## 비고
- `ctr`, `engagement_rate`는 성과 지표(타깃 변수 성격), 나머지는 콘텐츠 속성(피처 성격)으로 활용 가능.
- 중복 행(C003)과 결측치(C069, C070)는 분석 전 처리 방침을 decisions.md에 기록 권장.
